Plugin oficial del API de Cobru para ser usado con WooCommerce en Wordpress.

== Description ==

Somos Cobru - Pasarela de pago y billetera

Plugin oficial para recibir pagos en Wordpress + WooCommerce en minutos.

• Cobra
Acepta pagos fácil con tarjetas, efectivo, QR, PSE, Billeteras

• Envía
Envía dinero a cuentas de bancos, puntos de retiro en efectivo y billeteras

• Paga con QR
Paga fácil a amigos, comercios y sitios web usando nuestro código QR

• Depósito Cobru
Abre fácil y gratis tu depósito Cobru y empieza cobrar por internet

• Recarga
Recarga tu billetera Cobru usando efectivo, tarjetas

• Compra
Pines de videojuegos, recarga de celulares, seguros

• Acepta múltiple medios de pago
- PSE
- Tarjeta de crédito
- Efecty
- Corresponsal Bancolombia
- Botón Bancolombia
- Nequi
- Daviplata
- Rappipay
- Powwi
